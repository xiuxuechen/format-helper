# 抽取核对清单

- 统计段落、非空段落、表格、节数量。
- 检查 `word/document.xml` 是否存在 TOC 字段。
- 检查是否存在普通文本静态目录。
- 统计样式名称、字体、字号、加粗、缩进、行距、大纲级别。
- 识别总题目、一级标题、二级标题、三级标题、正文、表格标题，并输出角色置信度。
- 对 `1.`、`1.1`、`1.1.1` 等阿拉伯数字编号使用保守分类；无法结合上下文确认标题时标记为 `ambiguous-numbered-item`。
- 记录段落是否存在直接格式覆盖，以及段内 run 格式是否混合。
- 记录文档中可用样式 ID，供审计阶段判断目标原生样式是否存在。
- 记录横向节、页眉页脚、脚注、编号定义。
- 将无法确定的规则写入 `RULE_SUMMARY.md` 的人工确认项。
