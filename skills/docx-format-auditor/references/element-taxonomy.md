# 元素分类

| element_type | 含义 |
| --- | --- |
| `cover-title` | 总题目 |
| `cover-subtitle` | 封面副题名 |
| `cover-version` | 封面版本号 |
| `cover-org` | 封面编制单位或落款 |
| `cover-date` | 封面日期 |
| `toc-title` | 目录标题 |
| `toc-field` | Word 自动目录 |
| `toc-static-item` | 静态目录条目 |
| `heading-level-1` | 一级标题 |
| `heading-level-2` | 二级标题 |
| `heading-level-3` | 三级标题 |
| `heading-level-4` | 四级标题 |
| `ambiguous-numbered-item` | 编号形态可能是标题或列表项的低/中置信度段落 |
| `body-paragraph` | 正文段落 |
| `body-no-indent` | 无首行缩进正文 |
| `normal-table` | 普通表格 |
| `special-panel` | 专栏表格 |
| `appendix-table` | 附表 |
| `page-section` | 页面节 |
| `header-footer` | 页眉页脚 |
| `footnote` | 脚注 |
